# Hydro-Copula-DTC-BMA-Vine
Code and data for the paper "Complexity vs. Efficiency: Are Full-Structure Copula Models Worth the Cost in Multivariate Hydrological Simulation?". This repo benchmarks GtC, DTC, and BMA-Vine frameworks using synthetic and 3D/4D hydrological data. Includes scripts for CvM and Energy Distance evaluation to optimize model selection.
